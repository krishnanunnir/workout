    After

    Leave fresh

    Cooldown only if useful. Tomorrow matters more than today’s ego.

    Rep progression

    Same rounds. Slightly more reps.

    Progress if

    No pain, mild soreness, controlled form.

    If the habit starts to feel threatened, repeat the week.

    Reduce if

    Daily effort climbs past 6/10.

    Rest longer, use an easier variation, or do 2 rounds that day.

    Safety

    Stop for warning signs.

    Sharp pain, chest pain, dizziness, fainting, or unusual shortness of breath means stop.
