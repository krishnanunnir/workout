# Workout Routine Research

Current experiment: **Daily A/B — Bodyweight Foundation**.

This repo is for finding the right beginner workout fit. The current direction is a simple daily bodyweight plan:

- Every day
- Alternating A/B workouts
- No cardio included
- Bodyweight only
- Effort-based, not time-constrained
- Consistency over difficulty

## Files

- `AGENTS.md` — research-first repo instructions
- `plan.md` — written workout plan
- `web/` — visual version of the plan
- `scripts/serve.py` — local website server

## Website

Public site target:

```text
https://workout.bizzaretor.com
```

Open locally:

```bash
python3 scripts/serve.py
```

Then open:

```text
http://localhost:8765/web/
```
